Addepar Logos Download

Normal logo (default)

<div class="addelogo">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>

You can set the height of the fill to 100% to show it

<div class="addelogo">
	<div class="al-left"><div class="al-fill" style="height:100%;"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>

You can also set the height as a smaller percent.
For example, you may change this % to dynamically indicate downloading.

<div class="addelogo">
	<div class="al-left"><div class="al-fill" style="height:20%;"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>
			
<style type="text/css">
	@-webkit-keyframes 'loadIt'
		{       
		0% {
			height: 0%;
		}
		100% {
		  	height: 100%;
		}
	}

	.loadit .al-left .al-fill {
		-webkit-animation-name: loadIt;
		-webkit-animation-duration: 6s;
		-webkit-animation-iteration-count: infinite;
	}
</style>

<div class="addelogo loadit">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>

Hidden Logo, and animations

This could be used as a "loading" icon

<div class="addelogo hidden loadSync">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>

This one breaks up the logo...

<div class="addelogo hidden loading">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>

Only show animation once

Just add the "once" class.

<div class="addelogo hidden loadSync once">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>
