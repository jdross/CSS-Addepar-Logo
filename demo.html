Everything you need is Addelogo.css.

Normal logo (default)

The following four lines of HTML with addelogo.css will allow you to create logos anywhere.

The .addelogo element should always have a 2:3 height to width ratio.
<div class="addelogo">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>
You can set the height of the fill to 100% to show the blue
<div class="addelogo">
	<div class="al-left"><div class="al-fill" style="height:100%;"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>
You can also set the fill height as a smaller percent.
<div class="addelogo">
	<div class="al-left"><div class="al-fill" style="height:20%;"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>
For example, you may change this % to dynamically indicate downloading.
<style type="text/css">
	@-webkit-keyframes 'loadIt'
		{       
		0% {
			height: 0%;
		}
		100% {
		  	height: 100%;
		}
	}

	.loadit .al-left .al-fill {
		-webkit-animation-name: loadIt;
		-webkit-animation-duration: 5s;
		-webkit-animation-iteration-count: infinite;
		-webkit-animation-timing-function: linear;
	}
</style>

<div class="addelogo loadit">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>

Reversed logo

I put my thing down, flip it, and reverse it.

<div class="addelogo reversed">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>
				

<div class="addelogo reversed">
	<div class="al-left"><div class="al-fill" style="height:100%"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>

Hovers

Just add the "hover" class to the container.
<div class="addelogo hover">
	<div class="al-left"><div class="al-fill"></div></div>
	<div class="al-right"><div class="al-fill"></div></div>
</div>
